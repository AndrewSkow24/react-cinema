## useEffect

useEffect() - основная концепция реакта

```
fetch(ссылка на API)

https://jsonplaceholder.typicode.com/ — ссылка тестовых API


```

useEffect помещает в себе определённую стрелочную функцию

```
useEffect(()=>{}, [здесь можно указать при изменении каких переменных будет происходить побочный эффект])
```

пустой массив вторым аргументом означает, что вызов будет происходить только при первой прорисовке элемента, а не постоянно

```
return () => {clear}
```
